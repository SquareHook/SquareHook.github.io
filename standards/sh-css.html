SquareHook tool CSS Standards

  • Separate CSS into files named by the section and functionality
    Good:
    Navigation.css
    Footer.css
    ThemeAdjustments.css

    Bad: Custom.css
    Everything.css

  • Leave comments as necessary

  • Place media queries at the bottom of the file

  • Use Bootstrap Classes whenever possible

  • Avoid inline styling
